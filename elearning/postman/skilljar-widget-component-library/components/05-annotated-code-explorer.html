"model"
Model Selection

Specifies which Claude model to use. claude-opus-4-6 is Anthropic's most capable model, ideal for complex reasoning and nuanced tasks. You can swap this for claude-haiku-3-5 for faster, cheaper responses.

Required
"messages"
Conversation History

An array of message objects with role and content fields. Roles can be "user" or "assistant". Pass the full conversation history to maintain context across turns.

Required
"tools"
Available Tools

Declares which tools the model can invoke. Each tool has a name, description, and input_schema. Claude will decide when and which tool to call based on the user's request.

Optional
"max_tokens"
Output Token Limit

Sets the maximum number of tokens in the response. 4096 is a safe default for most tasks. For very long outputs (essays, code files), you may need to raise this. Higher values cost more.

Required
"temperature"
Randomness Control

Controls output variability from 0.0 (deterministic) to 1.0 (very creative). Use 0.0–0.3 for factual, consistent responses. Use 0.7–1.0 for brainstorming or creative writing.

Optional
"stream"
Streaming Mode

When true, the response is delivered as server-sent events (SSE), token by token. This makes the UI feel much faster. When false, the full response is returned in a single HTTP response.
